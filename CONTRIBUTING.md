# Beitragsrichtlinien

Vielen Dank für Ihr Interesse, zu diesem Projekt beizutragen!

## 🐛 Bug-Reports

Wenn Sie einen Fehler finden, erstellen Sie bitte ein [Issue](https://github.com/SaGa-ecm/Treppe/issues) mit einer möglichst genauen Beschreibung:
- Was ist passiert?
- Was haben Sie erwartet?
- Schritte zur Reproduktion
- Browser und Betriebssystem

## 💡 Feature-Requests

Haben Sie eine Idee für eine neue Funktion? Erstellen Sie ebenfalls ein Issue und beschreiben Sie Ihren Vorschlag.

## 🔧 Pull-Requests

Pull-Requests sind willkommen! Bitte beachten Sie:
1. Forken Sie das Repository.
2. Erstellen Sie einen neuen Branch für Ihre Änderung.
3. Halten Sie die Änderungen atomar (eine logische Änderung pro PR).
4. Stellen Sie sicher, dass der Code sauber und gut dokumentiert ist.

## 📧 Kontakt

Bei Fragen können Sie mich über [GitHub](https://github.com/SaGa-ecm) kontaktieren.
