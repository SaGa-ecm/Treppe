# Treppenplaner nach DIN 18065 & Eurocode 1

Ein browserbasiertes Planungstool für Holztreppen, das die Schrittmaßregel (DIN 18065) und Lastannahmen (Eurocode 1) berücksichtigt. Es unterstützt verschiedene Treppenformen, bietet einen Automatik- und einen manuellen Modus und ist für Laien und Fachleute gleichermaßen geeignet.

🌐 **Live-Demo:** [https://saga-ecm.github.io/Treppe/](https://saga-ecm.github.io/Treppe/)

## ✨ Features

- **Treppenformen:** Gerade, viertelgewendelt (90°), halbgewendelt (180°), Podesttreppe, Dachbodentreppe (Raumspar)
- **Berechnungsmodi:** Automatische Optimierung oder manuelle Eingabe von Auftritt, Steigung und Stufendicke
- **Normprüfungen:** Schrittmaß (2s+a = 59–65 cm), Steigung, Auftritt, Laufbreite, Nutzlast, Stufendicke
- **Lastannahmen:** Flächenlast und Einzellast in kN und kg (gemäß Eurocode 1)
- **Visualisierung:** Dynamische Canvas-Zeichnung (Seitenansicht / Draufsicht)
- **Benutzerfreundlichkeit:** Darkmode, Tooltips, Ampelsystem, responsive für mobile Geräte
- **Spendenformular:** PayPal-Integration mit frei wählbarem Betrag und Verwendungszweck

## 🛠️ Technologien

- HTML5
- CSS3 (Grid, Flexbox, CSS-Variablen für Darkmode)
- Vanilla JavaScript (ES6)
- GitHub Pages für Hosting

## 📖 Nutzung

1. Geben Sie die **Geschosshöhe** und die verfügbare **horizontale Entfernung** ein.
2. Wählen Sie die gewünschte **Treppenform**.
3. Passen Sie bei Bedarf die **manuellen Stufenmaße** an (Auftritt, Steigung, Stufendicke).
4. Das Tool berechnet live alle Werte und zeigt an, ob die Treppe den Normen entspricht (✅ = konform, ⚠️ = Warnung).
